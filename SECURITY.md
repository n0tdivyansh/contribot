# Security Policy

## Reporting

If you discover a vulnerability or a safety issue that could cause unintended repository modification, please report it privately to the maintainers before opening a public issue.

## Scope

Security for AICA includes:

- Command execution boundaries
- Authentication handling
- Unsafe repository mutation
- PR publishing behavior
- Dependency risks in the shipped CLI

## Expectations

Please include:

- Affected version
- Reproduction steps
- Impact assessment
- Suggested mitigation if available
